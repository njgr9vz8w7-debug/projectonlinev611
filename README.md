# Учитель Онлайн v4.5.0

Полнофункциональная система управления образовательным контентом с поддержкой групп/классов и календарем!

## ✨ Что нового в v4.5.0

### 👥 Система управления группами/классами
- **Создание групп** - разделение учеников по классам (10А, 10Б, и т.д.)
- **Управление членами** - добавление/удаление учеников из групп
- **Групповые назначения** - контент видит весь класс, а не одного ученика
- **Уведомления группам** - все члены получают оповещения при назначении

### 📅 Календарь (как в смартфоне)
- **Сетка календаря** - клеточки по дням, месяцам, годам
- **События на дни** - добавление событий на любую дату
- **Групповые события** - тесты, сроки, праздники для целого класса
- **Разные типы** - Задания, Тесты, Праздники, Напоминания
- **Просмотр месяца** - красивая сетка как в телефоне

### 🎯 Улучшенное назначение контента

#### Материалы (📄)
- Назначение всему классу/группе
- Видимость только для нужных учеников
- Отслеживание времени на материал

#### Новости (📰)
- Групповые новости
- Уведомления при публикации
- История по датам

#### Пособия (📚)
- Распределение по группам
- Доступ только для назначенных

#### Тесты (🧪)
- Назначение конкретным группам
- Сроки в календаре
- Результаты по группам

## 📋 API новых эндпоинтов v4.5.0

### Группы
- `POST /admin/groups/list` - список групп
- `POST /admin/groups/create` - создать группу
- `POST /admin/groups/add_member` - добавить ученика
- `POST /admin/groups/remove_member` - удалить ученика
- `POST /admin/groups/delete` - удалить группу

### Календарь
- `GET /calendar/<year>/<month>` - события за месяц
- `POST /calendar/add_event` - личное событие
- `POST /admin/calendar/add_event_group` - событие для группы
- `POST /calendar/delete_event/<id>` - удалить событие

### Назначение контента (обновлено)
- `POST /admin/materials/assign_group` - материал группе
- `POST /admin/news/assign_group` - новость группе
- `POST /admin/guides/assign_group` - пособие группе
- `POST /admin/tests/add_or_update` - тест с назначением
